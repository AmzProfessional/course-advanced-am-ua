### Модуль 3: SellerBoard, Аналітика та управління прибутковістю

**Мета модуля**: Ознайомити слухачів із ключовими функціями SellerBoard, навчити аналізувати дані магазину, виявляти проблеми та пропонувати рішення для підвищення прибутковості.

**Тривалість**: Приблизно 1 заняття (залежить від темпу викладання).

**Формат**: Уроки розбиті для зручності сприйняття, кожен фокусується на окремому аспекті SellerBoard.

### Урок 1: Вступ до SellerBoard та налаштування Dashboard

**Тривалість**: ~10 хвилин  
**Мета**: Познайомити з основними функціями SellerBoard та налаштуванням Dashboard для швидкого аналізу.

**Крок 1: Огляд SellerBoard**

- SellerBoard – основний інструмент для аналізу витрат, доходів, повернень, замовлень та інших метрик магазину.
- Переваги: об’єднує дані з різних звітів Amazon у зручний формат.

**Крок 2: Налаштування Dashboard**

- Dashboard – головний екран, де відображаються ключові показники.
- Налаштування плиток: вибір періоду (вчора, тиждень, місяць тощо) та параметрів (замовлення, продажі, витрати на рекламу, повернення, FBA fee, referral fee).
- Практична порада: налаштуйте Dashboard для щоденної перевірки за вчорашній день (замовлення, продажі, повернення, витрати).

**Крок 3: Практичне завдання**

- Відкрийте SellerBoard, налаштуйте Dashboard для відображення даних за вчорашній день. Визначте кількість проданих одиниць, витрати на рекламу та повернення.
