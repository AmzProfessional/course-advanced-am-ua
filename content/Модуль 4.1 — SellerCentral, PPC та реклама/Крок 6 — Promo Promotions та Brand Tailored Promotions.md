## Огляд кроку

**Мета**: Навчитися налаштовувати Promo Promotions і Brand Tailored Promotions у Seller Central для підвищення середнього чеку, трафіку та продажів.  

> **Примітка**: Перегляньте відео (2 хвилини), яке демонструє налаштування Promo Promotions і Social Media промокодів.  
> _[Місце для відео: вставте посилання на відео або вбудований плеєр у вашому Obsidian Vault.]_

---

## 1. Promo Promotions

> [!note] **Promo Promotions** — це акції в Seller Central (**Advertising** > **Promotions**) для стимулювання продажів через знижки або спеціальні пропозиції. Вони безкоштовні та підвищують середній чек і BSR.

### **Типи Promo Promotions**:

- **Percentage Off**: Знижка 5–15% за покупку кількох одиниць (наприклад, 2+ товари).
    - З 02.06.2025: включає Social Media промокоди через чекбокс.
- **Buy One Get One (BOGO)**: Купи один — отримай другий безкоштовно (або зі знижкою).
    - Ідеально для розпродажу стоку, впливає на BSR.

### **Social Media промокоди**:

- Знижка: 15–25%.
- Використовуються інфлюенсерами для залучення трафіку.
- Популярні в нішах beauty, кухонних товарів, електроніки.
- Налаштування: увімкніть чекбокс Social Media у Percentage Off (з 02.06.2025).

> [!note]  **Promo Promotions** – Percentage Off (5–15%, включає Social Media) і Buy One Get One, безкоштовні.

### **Коли використовувати**:

- Для підвищення середнього чеку (Percentage Off за 2+ одиниці).
- Для розпродажу стоку (BOGO).
- Для залучення трафіку через інфлюенсерів (Social Media промокоди).

**Приклад**: Percentage Off 7% за покупку “Golden Bracelet” і “Silver Bracelet” (AquaFit Store) підвищив середній чек на 15% (з $20 до $23 за замовлення).

> **Візуалізація**:  
> _Місце для скріншота_: Додайте зображення Seller Central із налаштуванням Percentage Off для двох ASIN.  
> _Опис_: Покажіть, як виглядає інтерфейс Promo Promotions.

## 2. Brand Tailored Promotions

**Brand Tailored Promotions** — це цільові знижки для певних груп покупців, доступні лише для зареєстрованих брендів у Seller Central (**Advertising** > **Promotions**). Вони підвищують конверсію та лояльність.

### **Типи аудиторій**:

- Нові клієнти (ніколи не купували ваш бренд).
- Покупці, які додавали товар у кошик, але не купили.
- Повторні покупці (купували раніше).

### **Особливості**:

- Знижка: 5–20% (налаштовується).
- Період: 1–30 днів.
- Лише для брендів із Brand Registry.
- Безкоштовні, але потребують узгодження з менеджером.

### **Коли використовувати**:

- Для залучення нових клієнтів.
- Для повернення тих, хто покинув кошик.
- Для стимулювання повторних покупок.

**Приклад**: Brand Tailored Promotion (10% для нових клієнтів) на “Golden Bracelet” залучила 5 нових покупців за 7 днів, підвищивши продажі на 10%.

> [!note]  **Brand Tailored Promotions** – знижки для цільової аудиторії, лише для брендів.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте налаштування Brand Tailored Promotion із вибором аудиторії.  
> _Опис_: Покажіть, як обрати цільову групу в Seller Central.

## 3. Інтерактивний чекліст

Використовуйте цей чекліст у Obsidian для відстеження прогресу:

- [ ]  Відкрили **Promotions** і вибрали ASIN “Golden Bracelet” та споріднений.
- [ ]  Налаштували Percentage Off: 7%, 2+ одиниці, 7 днів, Social Media чекбокс.
- [ ]  Підготували заявку на Social Media промокод: 15%, ніша, ціль.
- [ ]  Склали звіт із налаштуваннями та рекомендаціями.
- [ ]  Узгодили звіт із менеджером (**Freelance** > **Заявка**).

## 4. Ключові поради для Account Manager’ів

- **Percentage Off**: Використовуйте 5–10% для економії маржинальності.
- **Social Media**: Обирайте інфлюенсерів із цільовою аудиторією (beauty, кухня).
- **Brand Tailored**: Перевіряйте Brand Registry перед налаштуванням.
- **Заявки**: Скидайте скріншоти/текст на апрув старшому менеджеру.
- **Моніторинг**: Перевіряйте середній чек і продажі після акції (**Business Reports** > **Sales Dashboard**).

> **Плашка**:  
> Не створюйте заявки самостійно без апруву старшого менеджера. Скидайте весь контент заявки (тексти, скріншоти) на перевірку.

> **Візуалізація**:  
> _Місце для GIF_: Додайте GIF (3–5 секунд), що показує налаштування Percentage Off із Social Media чекбоксом.  
> _Опис_: Демонстрація, як створити акцію в Seller Central.
