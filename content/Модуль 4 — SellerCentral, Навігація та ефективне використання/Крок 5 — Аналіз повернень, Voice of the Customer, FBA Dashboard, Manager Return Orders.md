## 1. Voice of the Customer: Аналіз причин повернень

**Voice of the Customer** — це розділ у Seller Central, який показує причини повернень за ASIN, отримані з коментарів клієнтів і Performance Notifications (з Уроку 2).

![[096e240b-d482-4074-ba2d-3f304b1d97dd.png]]
### Що перевіряти:

- **Причини повернень**: Помилки комплектації, невідповідність опису, дефекти товару.
- **Коментарі клієнтів**: Деталі скарг (наприклад, "негерметичний" чи "не той розмір").
- **Тренди**: Повторювані проблеми, які потребують змін у лістингу.

### Дії:

1. У Seller Central перейдіть до **Performance** > **Voice of the Customer**.
2. Виберіть ASIN і перегляньте таблицю з причинами повернень.
3. Проаналізуйте коментарі клієнтів у стовпці **Customer Comments**.
4. Виявіть повторювані проблеми (наприклад, "негерметичний").
5. Створіть заявку у внутрішній платформі (**Freelance** > **SEO** або **Саппорт**) для оновлення лістингу або зверніться до техпідтримки Amazon.

**Приклад**: Для ASIN "Bottle Holder" у Voice of the Customer 5 повернень через "негерметичність". Створіть заявку для SEO, щоб додати "leak-proof" до тайтла.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте зображення Voice of the Customer із таблицею повернень і коментарями.  
> _Опис_: Покажіть, як виглядає аналіз причин повернень.

## 2. FBA Dashboard: Статистика повернень

**FBA Dashboard** — це інструмент у Seller Central, який показує статистику повернень, їхній вплив на прибуток і додаткові комісії за повернення.

![[il_fullxfull.2303734266_essv.jpg]]
### Що перевіряти:

- **Топ товарів із поверненнями**: ASIN із найвищим відсотком повернень.
- **Відсоток повернень**: Порівняйте з середнім по категорії (наприклад, 8.7% для Home & Kitchen).
- **Комісії**: Додаткові витрати за повернення (наприклад, $3.70 за одиницю).

### Дії:

1. У Seller Central перейдіть до **Reports** > **Fulfillment by Amazon** > **FBA Dashboard**.
2. Перегляньте розділ **Customer Returns**, відсортуйте за відсотком повернень.
3. Порівняйте відсоток повернень вашого ASIN із середнім по категорії.
4. Якщо відсоток вищий (наприклад, >10%), проаналізуйте причини в Voice of the Customer.
5. Запропонуйте дії (оновлення лістингу, купон) або створіть заявку (**Freelance** > **SEO**).

**Приклад**: Для "Bottle Holder" відсоток повернень 12% (вище середнього 8.7% для Home). Запропонуйте оновити опис із деталями про герметичність.

> **Візуалізація**:  
> _Місце для GIF_: Додайте GIF (3-5 секунд), який показує сортування повернень у FBA Dashboard.  
> _Опис_: Демонстрація, як аналізувати статистику повернень.

## 3. Manage Return Orders: Детальний аналіз повернень

**Manage Return Orders** — це розділ у Seller Central для перегляду списку повернень із детальними коментарями клієнтів, що допомагає коригувати лістинги.

![[Pasted image 20250609111756.png]]
### Що перевіряти:

- **Список повернень**: Дата, ASIN, причина повернення, коментарі клієнтів.
- **Типові причини**: Наприклад, "не підходить розмір" або "дефектний товар".

### Дії:

1. У Seller Central перейдіть до **Orders** > **Manage Return Orders**.
2. Відфільтруйте повернення за ASIN і періодом (наприклад, останній місяць).
3. Перегляньте коментарі клієнтів у стовпці **Reason for Return**.
4. Якщо проблема повторюється (наприклад, "не той розмір"), оновіть опис/розміри в лістингу.
5. Створіть заявку (**Freelance** > **SEO** або **Саппорт**) для виправлення або зверніться до старшого менеджера.

**Приклад**: У Manage Return Orders для "Bottle Holder" 3 повернення через "не підходить розмір". Додайте таблицю розмірів до опису лістингу.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте зображення Manage Return Orders із коментарями клієнтів.  
> _Опис_: Покажіть, як виглядає список повернень.

## 4. Практичне завдання

**Завдання**: Проаналізуйте повернення та запропонуйте рішення.  
**Інструкція**:

1. У Seller Central відкрийте **Performance** > **Voice of the Customer**.
    - Виберіть ASIN (наприклад, "Bottle Holder") і визначте основну причину повернень.
2. Перейдіть до **Reports** > **Fulfillment by Amazon** > **FBA Dashboard**.
    - Перевірте відсоток повернень для ASIN і порівняйте з середнім по категорії (наприклад, 8.7%).
3. Відкрийте **Orders** > **Manage Return Orders**, відфільтруйте за ASIN.
    - Проаналізуйте коментарі клієнтів і виявіть повторювані проблеми.
4. Запропонуйте рішення (наприклад, оновлення тайтла, опису, запуск купона).
5. Підготуйте звіт:
    - Опишіть причину повернень і відсоток.
    - Вкажіть запропоноване рішення та дії (заявка, зміна лістингу).

**Результат**:

- Зробіть скріншоти Voice of the Customer, FBA Dashboard і Manage Return Orders.
    - ASIN товару.
    - Причина повернень (наприклад, "негерметичність").
    - Відсоток повернень (наприклад, 12% проти 8.7%).
    - Пропозиція для старшого менеджера (наприклад, "Додати 'leak-proof' до тайтла, створити заявку для SEO").

> **Візуалізація**:  
> _Місце для скріншота_: Додайте звіт із причинами повернень і пропозиціями.  
> _Опис_: Покажіть, як виглядає аналіз і звіт.

## 6. Ключові поради для Account Manager'ів

- **Регулярність**: Перевіряйте повернення щотижня для топових ASIN.
- **Пріоритетність**: Зосередьтеся на товарах із відсотком повернень вище середнього по категорії.
- **Проактивність**: Виявляйте повторювані проблеми та пропонуйте зміни в лістингах.
- **Заявки**: Використовуйте **Freelance** для SEO (оновлення лістингів) або сапорту (звернення до Amazon).
- **Комунікація**: Узгоджуйте рішення зі старшим менеджером, щоб уникнути штрафів.
