#### Урок 5: Аналіз повернень: Voice of the Customer, FBA Dashboard, Manager Return Orders

**Тривалість**: ~7 хвилин  
**Зміст**:

- Voice of the Customer: аналіз причин повернень через ASIN із Performance Notifications.
    - Плашка: **Voice of the Customer** – розділ для виявлення причин повернень (наприклад, помилки комплектації чи невідповідність).
    - Дії: аналіз коментарів клієнтів, виявлення повторюваних проблем, передача в техпідтримку Amazon.
- FBA Dashboard: статистика повернень (топ товарів, причини, відсоток).
    - Плашка: **FBA Dashboard** – статистика повернень, що показує вплив на прибуток і комісії.
    - Дії: порівняння відсотка повернень із середнім по категорії (наприклад, 8.7% для Home), уникнення додаткових комісій (наприклад, $3.70 за одиницю).
- Manager Return Orders: перегляд списку повернень із коментарями клієнтів.
    - Плашка: **Manager Return Orders** – детальний аналіз повернень для коригування листингів.
    - Дії: перевірка коментарів (наприклад, «не підходить розмір»), оновлення опису чи розмірів.
- Практика: аналіз повернення в Voice of the Customer за ASIN, перевірка відсотка повернень у FBA Dashboard, перегляд коментарів у Manager Return Orders.  
    **Мета уроку**: Навчити аналізувати повернення для зменшення їх відсотка та уникнення штрафів.
