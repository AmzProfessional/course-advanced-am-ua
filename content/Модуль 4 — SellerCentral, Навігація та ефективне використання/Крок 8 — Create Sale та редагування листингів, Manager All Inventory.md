#### Урок 8: Create Sale та редагування листингів: Manager All Inventory

**Тривалість**: ~7 хвилин  
**Зміст**:

- Create Sale: безкоштовна акція для надлишкових товарів.
    - Плашка: **Create Sale** – безкоштовна акція на 14 днів у Today’s Deals для розпродажу надлишків.
    - Дії: перевірка реклами, запуск акції за ціною Amazon (наприклад, зниження з $29.99 до $20.04).
- Manager All Inventory: редагування листингів.
    - Плашка: **Manager All Inventory** – розділ для редагування статусів, цін, описів і атрибутів листингів.
    - Дії: оновлення атрибутів (наприклад, вага, розміри), цін, фотографій, відключення/видалення листингів.
- Практика: запуск Create Sale для товару з надлишками, редагування атрибутів листинга в Manager All Inventory.  
    **Мета уроку**: Навчити використовувати акції та редагувати листинги для підвищення продажів і оптимізації запасів.
