> **Примітка**: Перегляньте відео (2.5 хвилини), яке демонструє запуск акції Create Sale та редагування лістингів у Seller Central.

## 1. Create Sale: Безкоштовна акція для надлишкових товарів

**Create Sale** — це інструмент у Seller Central для створення безкоштовних акцій на 14 днів у розділі Today’s Deals, спрямованих на розпродаж надлишкових товарів (наприклад, з Edge Inventory, Урок 7).

### Що перевіряти:

- **Надлишкові товари**: ASIN із низьким Sales Row (<10 одиниць за 90 днів, Урок 6) або в Edge Inventory (>181 день).
- **Ціна акції**: Знижка, рекомендована Amazon (наприклад, з $29.99 до $20.04).
- **Реклама**: Наявність активних PPC-кампаній для посилення акції.

### Дії:

1. У Seller Central перейдіть до **Pricing** > **Create Sale**.
2. Виберіть ASIN із надлишками (перевірте в **Edge Inventory** або **BeInventory**).
3. Встановіть знижку, рекомендовану Amazon (наприклад, $20.04), або узгодьте зі старшим менеджером.
4. Перевірте, чи активна PPC-кампанія для ASIN (**Advertising** > **Campaign Manager**).
5. Запустіть акцію на 14 днів, відстежуйте продажі в **Reports** > **Business Reports**.
6. Якщо акція не ефективна, створіть заявку (**Freelance** > **PPC** або **Саппорт**).

**Приклад**: Для ASIN "Bottle Holder" 100 одиниць у Edge Inventory, Sales Row = 5 за 90 днів. Запустіть Create Sale зі знижкою з $29.99 до $20.04 і перевірте PPC-кампанію.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте зображення Create Sale із вибраним ASIN і рекомендованою знижкою.  
> _Опис_: Покажіть, як виглядає налаштування акції.

## 2. Manage All Inventory: Редагування листингів

**Manage All Inventory** — це розділ у Seller Central для редагування атрибутів, цін, описів і статусів лістингів, що допомагає виправляти проблеми (наприклад, Search Suppressed, Урок 4) і підвищувати продажі.

### Що редагувати:

- **Атрибути**: Вага, розміри, матеріали, ключові слова.
- **Ціни**: Оновлення для конкуренції за Buy Box (з Уроку 4).
- **Описи та фото**: Додавання детальної інформації чи нових зображень.
- **Статуси**: Відключення (Close) або видалення (Delete) лістингів із низькими продажами.

### Дії:

1. У Seller Central перейдіть до **Inventory** > **Manage All Inventory**.
2. Знайдіть ASIN (наприклад, через пошук або фільтр **Active Listings**).
3. Натисніть **Edit** для редагування:
    - Оновіть атрибути (наприклад, додайте вагу "0.5 lbs" або розміри "10x5x3 inches").
    - Змініть ціну (узгодьте зі старшим менеджером).
    - Додайте фото чи оновіть опис (наприклад, додайте "leak-proof").
4. Для непродаваних лістингів виберіть **Close** або **Delete** (після узгодження).
5. Якщо потрібні зміни (SEO, фото), створіть заявку (**Freelance** > **SEO** або **Дизайн**).

**Приклад**: Для ASIN "Bottle Holder" статус Search Suppressed через відсутність розмірів. Додайте розміри "10x5x3 inches" і оновіть опис із "leak-proof".

> **Візуалізація**:  
> _Місце для GIF_: Додайте GIF (3-5 секунд), який показує редагування атрибутів і ціни в Manage All Inventory.  
> _Опис_: Демонстрація, як оновити лістинг.

## 3. Практичне завдання

**Завдання**: Запустіть Create Sale і відредагуйте лістинг для ASIN із надлишками.  
**Інструкція**:

1. У Seller Central відкрийте **Pricing** > **Create Sale**.
    - Виберіть ASIN із надлишками (перевірте **Edge Inventory** або **BeInventory**).
    - Встановіть знижку, рекомендовану Amazon (наприклад, з $29.99 до $20.04).
    - Перевірте наявність PPC-кампанії в **Campaign Manager**.
2. Перейдіть до **Inventory** > **Manage All Inventory**, знайдіть той самий ASIN.
    - Оновіть атрибути (наприклад, розміри "10x5x3 inches").
    - Додайте до опису ключові слова (наприклад, "leak-proof").
    - Якщо лістинг непродаваний, запропонуйте **Close** або **Delete**.
3. Підготуйте звіт:
    - Опишіть акцію Create Sale (знижка, PPC).
    - Вкажіть зміни в лістингу (атрибути, опис).
    - Пропонуйте дії (заявка, видалення лістингу).

**Результат**:

- Зробіть скріншоти Create Sale (налаштування акції) і Manage All Inventory (редагування лістингу).
- Запишіть у нотатки в Obsidian:
    - ASIN товару.
    - Дані акції (знижка з $29.99 до $20.04, PPC-кампанія).
    - Зміни в лістингу (розміри, опис).
    - Пропозиція для старшого менеджера (наприклад, "Запустити акцію, оновити лістинг, створити заявку для SEO").

> **Візуалізація**:  
> _Місце для скріншота_: Додайте звіт із деталями акції та редагування лістингу.  
> _Опис_: Покажіть, як виглядає аналіз і пропозиція.
