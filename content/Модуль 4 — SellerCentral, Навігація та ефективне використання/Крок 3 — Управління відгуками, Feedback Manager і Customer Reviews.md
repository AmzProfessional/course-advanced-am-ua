## Огляд кроку

**Мета**: Навчитися обробляти відгуки в Feedback Manager і Customer Reviews, видаляти негативні відгуки про продавця та підвищувати лояльність клієнтів через рефанди й відповіді. 

> **Примітка**: Перегляньте відео (2.5 хвилини), яке демонструє роботу з Feedback Manager і Customer Reviews у Seller Central.  
> _[Місце для відео: вставте посилання на відео або вбудований плеєр у вашому Obsidian Vault.]_

## 1. Feedback Manager: Відгуки про продавця

**Feedback Manager** — це розділ у Seller Central, де відображаються відгуки про продавця, які впливають на Buy Box і репутацію аккаунта. Позитивні відгуки (4-5 зірок) підвищують довіру, а негативні (1-3 зірки) можуть зашкодити продажам.

### Що робити:

- **Позитивні відгуки (4-5 зірок)**: Залишайте без змін, вони покращують метрики.
- **Негативні відгуки (1-3 зірки)**:
    - Для FBA-товарів: Видаляйте через **Request Removal**, оскільки Amazon автоматично обробляє такі запити (FBA відповідає за доставку).
    - Для FBM-товарів: Аналізуйте причину, зв’яжіться з клієнтом або створіть заявку для сапорту.

### Як видалити негативний відгук:

1. У Seller Central перейдіть до **Performance** > **Feedback Manager**.
2. Знайдіть відгук із рейтингом 1-3 зірки для FBA-товару.
3. Натисніть **Request Removal** і підтвердіть запит.
4. Перевіряйте статус через 3-5 днів (Amazon зазвичай видаляє FBA-відгуки автоматично).

**Приклад**: Клієнт залишив 2-зірковий відгук про "Bottle Holder" через затримку доставки (FBA). Використайте **Request Removal**, щоб видалити відгук.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте зображення Feedback Manager із негативним відгуком і кнопкою Request Removal.  
> _Опис_: Покажіть, як виглядає процес видалення відгуку.

## 2. Customer Reviews: Відгуки про товари

**Customer Reviews** — це відгуки про товари, які впливають на лояльність клієнтів і рішення про покупку. На відміну від Feedback Manager, ці відгуки не можна видаляти, крім випадків із образливим вмістом.

### Що робити:

- **Позитивні відгуки (4-5 зірок)**: Залишайте без змін, вони підвищують конверсію.
- **Негативні відгуки (1-3 зірки)**:
    - Пропонуйте рефанд через **Offer Refund** у Seller Central, щоб вирішити проблему клієнта.
    - Відповідайте клієнту публічно протягом 24 годин, використовуючи шаблонні відповіді.
    - Якщо відгук образливий, створіть заявку в Seller Central для видалення.

### Як обробити негативний відгук:

1. У Seller Central перейдіть до **Performance** > **Customer Reviews**.
2. Знайдіть відгук із рейтингом 1-3 зірки для ASIN.
3. Натисніть **Offer Refund**, запропонуйте повернення коштів (затверджуйте зі старшим менеджером).
4. Напишіть відповідь клієнту (наприклад, "Вибачте за незручності, ми відправили рефанд").
5. Якщо відгук образливий, подайте запит на видалення через **Contact Seller Support**.

**Приклад**: Клієнт залишив 1-зірковий відгук для "Bottle Holder", скаржачись на негерметичність. Пропонуйте рефанд і відповідайте: "Вибачте, ми відправили рефанд. Зв’яжіться з нами для заміни."

> **Візуалізація**:  
> _Місце для GIF_: Додайте GIF (5 секунд), який показує пропозицію рефанду та відповідь клієнту в Customer Reviews.  
> _Опис_: Демонстрація, як обробити відгук про товар.

## 3. Практичні поради та шаблонні відповіді

- **Feedback Manager**:
    - Перевіряйте відгуки щотижня, видаляйте негативні FBA-відгуки негайно.
    - Для FBM-відгуків створюйте заявки у внутрішній платформі (**Freelance** > **Саппорт**).
- **Customer Reviews**:
    - Відповідайте на негативні відгуки протягом 24 годин, щоб показати турботу.
    - Пропонуйте рефанди лише після узгодження зі старшим менеджером.
    - Для образливих відгуків створюйте заявку через Seller Central.
- **Шаблонні відповіді** (Customer Reviews):
    - "Вибачте за незручності! Ми відправили рефанд. Будь ласка, зв’яжіться з нами для вирішення."
    - "Дякуємо за відгук! Ми запропонували рефанд і готові допомогти з заміною товару."

**Приклад**: Для ASIN "Bottle Holder" видаліть 2-зірковий FBA-відгук у Feedback Manager і запропонуйте рефанд для 1-зіркового відгуку в Customer Reviews із відповіддю за шаблоном.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте приклад шаблонної відповіді в Customer Reviews.  
> _Опис_: Покажіть, як виглядає відповідь клієнту.

## 4. Практичне завдання

**Завдання**: Обробіть відгуки в Feedback Manager і Customer Reviews.  
**Інструкція**:

1. Увійдіть у Seller Central, відкрийте **Performance** > **Feedback Manager**.
2. Знайдіть негативний відгук (1-3 зірки) для FBA-товару.
    - Використайте **Request Removal** для видалення.
3. Перейдіть до **Customer Reviews**, знайдіть негативний відгук (1-3 зірки) для ASIN.
    - Пропонуйте рефанд через **Offer Refund** (затвердьте зі старшим менеджером).
    - Напишіть відповідь клієнту за шаблоном протягом 24 годин.
4. Якщо відгук образливий, створіть заявку через **Contact Seller Support**.
5. Підготуйте звіт:
    - Опишіть дії з відгуками (видалення, рефанд, відповідь).
    - Вкажіть ASIN і статус обробки.

**Результат**:

- Зробіть скріншоти Feedback Manager (видалення відгуку) і Customer Reviews (рефанд/відповідь).
- Запишіть у нотатки в Obsidian:
    - ASIN товару.
    - Опис відгуків (Feedback Manager і Customer Reviews).
    - Дії (видалення, рефанд, відповідь).
    - Пропозиція для старшого менеджера (наприклад, "Видалено FBA-відгук, запропоновано рефанд").

> **Візуалізація**:  
> _Місце для скріншота_: Додайте звіт із діями та скріншотами відгуків.  
> _Опис_: Покажіть, як виглядає обробка відгуків.

- [ ]  Перевірено Customer Reviews на негативні відгуки.
- [ ]  Запропоновано рефанд і написано відповідь клієнту.
- [ ]  Створено заявку для образливого відгуку (якщо є).
- [ ]  Підготовлено звіт для менеджера.

- **Пріоритетність**: Видаляйте негативні FBA-відгуки в Feedback Manager якомога швидше.

- **Шаблони**: Використовуйте стандартні відповіді для економії часу.
- **Координація**: Узгоджуйте рефанди зі старшим менеджером, щоб уникнути збитків.
- **Моніторинг**: Перевіряйте відгуки щотижня, особливо для топових товарів.
