## 1. Inventory Performance Index (IPI): Рейтинг управління запасами

**Inventory Performance Index (IPI)** — це рейтинг Amazon, який відображає ефективність управління запасами. IPI впливає на ліміт дозволеного обсягу складу Амазон.

### Складові IPI:

- **Excess Inventory**: Надлишкові запаси (>90 днів на складі).
- **FBA sell-through**: Обертовість товару (продажі за 90 днів).
- **Stranded Inventory**: Недоступні для продажутовари, в яких завис сток (наприклад, через деактивовані лістинги).
- **FBA In-Stock**: Наявність стоку (запаси на 30-60 днів).

![[Screen-Shot-2023-10-23-at-3.17.37-PM.avif]]
### Що перевіряти:

- **Зелена зона**: IPI >400 (без обмежень на склад).
- **Червона зона**: IPI <400 (обмеження складських лімітів, додаткові комісії).
- **Червона зона складових частин**: будь який показник червоним потребую реакції.
- **Stranded Inventory**: Обов’язково перевіряти весь застряглий сток, а також причини чому товар туди потрапив. Аналіз передавати старшому менеджеру.
### Дії:

1. У Seller Central перейдіть до **Inventory** > **Inventory Dashboard**.
2. Перевірте IPI на головній сторінці (наприклад, 420 = зелена зона).
3. Якщо IPI <400, проаналізуйте складові (Excess, Sales Row тощо) у **Inventory Performance**.
4. Створіть заявку (**Freelance** > **SEO** або **PPC**) для виправлення проблем (наприклад, запуск реклами для низьких продажів).

**Приклад**: IPI = 380 (червона зона) через високий Excess Inventory для ASIN "Bottle Holder". Запропонуйте знижку або рекламу для розпродажу надлишків.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте зображення Inventory Dashboard із показником IPI.  
> _Опис_: Покажіть, як виглядає IPI та його складові.

## 2. Ключові поради для Account Manager'ів

- **Регулярність**: Перевіряйте IPI щодня під час перевірки акк хелс.
- **Пріоритетність**: Зосередьтеся на Low Stock (<28 днів) і Excess Inventory (>90 днів).
