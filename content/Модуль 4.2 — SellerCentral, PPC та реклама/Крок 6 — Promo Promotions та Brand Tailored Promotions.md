**Мета**: Навчитися налаштовувати Promotions і Brand Tailored Promotions у Seller Central для підвищення середнього чеку, трафіку та продажів.

## 1. Promotions

> [!note] **Promotions** — це акції в Seller Central (**Advertising** > **Promotions**) для стимулювання продажів через знижки або спеціальні пропозиції. Вони безкоштовні та підвищують середній чек і BSR.

![[Pasted image 20250915230049.png]]
### **Типи Promotions**:

- **Percentage Off**: Знижка 5–15% за покупку кількох одиниць (наприклад, 2+ товари).
    - Включає Social Media промокоди через чекбокс.
- **Buy One Get One (BOGO)**: Купи один — отримай другий безкоштовно (або зі знижкою).
    - Ідеально для розпродажу стоку, впливає на BSR.

### **Social Media промокоди**:

- Знижка: 15–25%.
- Використовуються інфлюенсерами для залучення трафіку.
- Популярні в нішах beauty, кухонних товарів, електроніки.
- Налаштування: увімкніть чекбокс Social Media у Percentage Off.

> [!note]  **Promotions** – Percentage Off (5–15%, включає Social Media) і Buy One Get One, безкоштовні.

### **Коли використовувати**:

- Для підвищення середнього чеку (Percentage Off за 2+ одиниці).
- Для розпродажу стоку (BOGO).
- Для залучення трафіку через інфлюенсерів (Social Media промокоди).

**Приклад**: Percentage Off 7% за покупку “Golden Bracelet” і “Silver Bracelet” (AquaFit Store) підвищив середній чек на 15% (з 20 доларів до 23 за замовлення).

## 2. Brand Tailored Promotions

> [!note] **Brand Tailored Promotions** — це цільові знижки для певних груп покупців, доступні лише для зареєстрованих брендів у Seller Central (**Advertising** > **Brand Tailored Promotions**). Вони підвищують конверсію та лояльність.
### **Типи аудиторій**:

- Покупці, які додавали товар у кошик, але не купили.
- Повторні покупці (купували раніше).
### **Особливості**:

- Знижка: 5–20% (налаштовується).
- Період: 1–30 днів.
- Лише для брендів із Brand Registry.
- Безкоштовні.
### **Коли використовувати**:

- Для повернення тих, хто покинув кошик.
- Для стимулювання повторних покупок.

> [!note]  **Brand Tailored Promotions** – знижки для цільової аудиторії, лише для брендів.

---
## 3. Ключові поради для Account Manager’ів

- **Percentage Off**: Використовуйте 5–10% для економії маржинальності.
- **Social Media**: Обирайте інфлюенсерів із цільовою аудиторією (beauty, кухня).
- **Brand Tailored**: Перевіряйте Brand Registry перед налаштуванням.
- **Заявки**: Скидайте скріншоти/текст на апрув старшому менеджеру.
- **Моніторинг**: Перевіряйте середній чек і продажі після акції.
