## Огляд кроку

**Мета**: Навчитися запускати акції через Create outlet deal для розпродажу надлишкових товарів і редагувати лістинги в Manage All Inventory для підвищення продажів та оптимізації запасів.  

---
## 1. Create Sale: Безкоштовна акція для надлишкових товарів

> [!note] **Create Sale** — це інструмент у Seller Central для створення безкоштовних акцій на 14 днів у розділі Today’s Deals, спрямованих на розпродаж надлишкових товарів.
### Що перевіряти:

- **Надлишкові товари**: ASIN із низьким Sales Row (<10 одиниць за 90 днів, Урок 6) або в Edge Inventory (>181 день).
- **Ціна акції**: Знижка, рекомендована Amazon (наприклад, з 29.99 доларів до 20.04 доларів).
### Дії:
1. У Seller Central перейдіть до FBA Inventory
2. Поставте фільтр по Recommendation / Create outlet deal.  
3. Перепровірте сток по цим товарам, особливу увагу зверніть на сток більше ніж 180 днів
4. Натисніть Create outlet deal, перегляньте на які дати буде запущена знижка та яку ціну пропонує Амазон.  
5. Зробіть розрахунок, якщо він вас влаштовує - передайте старшому менеджеру з аргументами чому саме тут варто запустити outlet deal.
---

## 2. Manage All Inventory: Редагування листингів

> [!note] **Manage All Inventory** — це розділ у Seller Central для редагування атрибутів, цін, описів і статусів лістингів, що допомагає виправляти проблеми (наприклад, Search Suppressed, Урок 4) і підвищувати продажі.

### Що редагувати:

- **Атрибути**: Вага, розміри, матеріали, ключові слова.
- **Ціни**: Оновлення для конкуренції за Buy Box.
- **Описи та фото**: Додавання детальної інформації чи нових зображень.
### Дії:

1. У Seller Central перейдіть до **Inventory** > **Manage All Inventory**.
2. Знайдіть ASIN (наприклад, через пошук або фільтр **Active Listings**).
3. Натисніть **Edit** для редагування:
    - Оновіть атрибути (наприклад, додайте вагу "0.5 lbs" або розміри "10x5x3 inches").
    - Змініть ціну (узгодьте зі старшим менеджером).
    - Додайте фото чи оновіть опис (наприклад, додайте "leak-proof").

**Приклад**: Для ASIN "Bottle Holder" статус Search Suppressed через відсутність розмірів. Додайте розміри "10x5x3 inches" і оновіть опис із "leak-proof".

## 3. Ключові поради для Account Manager'ів

- **Обережність із цінами**: Узгоджуйте знижки в Create Sale зі старшим менеджером.
- **Оптимізація**: Оновлюйте атрибути та описи для уникнення Search Suppressed.
- **Заявки**: Використовуйте **Freelance** для PPC (реклама), SEO (описи) або дизайну (фото).
- **Моніторинг**: Відстежуйте продажі після акції.
