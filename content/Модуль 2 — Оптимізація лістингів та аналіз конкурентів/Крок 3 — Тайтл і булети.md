## 1. Тайтл: Оптимізація для SEO

**Тайтл** — це заголовок листинга, який містить ключові слова для пошуку та основні характеристики товару. Він впливає на видимість у пошуку Amazon і привабливість для клієнтів.

### **Вимоги до тайтла**:

- **Довжина**: До 200 символів (включаючи пробіли).
- **Зміст**: Бренд, назва товару, ключові слова (наприклад, матеріал, колір), характеристики (розміри, функціонал).
- **SEO**: Включає популярні пошукові запити, але уникає дублювання слів (щоб Amazon не скоротив тайтл).
- **Чіткість**: Зрозумілий для клієнтів, без стоп-слів (згідно з правилами Amazon).

![[content/images/15.gif]]
### Що перевіряти:

- Довжину тайтла (≈200 символів).
- Наявність ключових слів (перевірте через Seller Sprite).
- Відсутність змін, внесених Amazon (наприклад, скорочення через дублювання слів).

### Дії:

1. У Seller Central перейдіть до **Inventory** > **Manage All Inventory**.
2. Перевірте тайтл: підрахуйте символи (використовуйте текстовий редактор або Seller Sprite).
3. Якщо тайтл коротший за 180 символів або містить дублювання, створіть заявку (**Freelance** > **SEO**) для оновлення або надішліть пропозицію старшому менеджеру.
4. Повідомте старшого менеджера, якщо Amazon змінив тайтл.

**Приклад**: Для ASIN "Bottle Holder" тайтл: "AquaFit Bottle Holder, Stainless Steel, Gold, 32 oz, Leak-Proof, BPA-Free, Insulated for Hot/Cold Drinks" (120 символів). Якщо скорочено до 80 символів, замовте SEO-оптимізацію.

> **Візуалізація**:  
> _Місце для скріншота_: Додайте зображення тайтла в Seller Central (Vital Info) та на лістингу.  
> _Опис_: Покажіть, як виглядає перевірка тайтла.

## 2. Буллети: Мотивація до покупки

**Буллети** — це 5 коротких пунктів у лістингу, які описують переваги, матеріали, функціонал і характеристики товару, мотивуючи клієнтів до покупки.

### **Вимоги до булетів**:

- **Кількість**: 5 пунктів (рідко 6 за винятком).
- **Довжина**: Кожен пункт до 250 символів, короткий і чіткий.
- **Зміст**: Переваги (наприклад, герметичність), матеріали (нержавіюча сталь), розміри, функціонал (зберігає температуру).
- **SEO**: Містять ключові слова, але уникають дублювання з тайтлом.

### Що перевіряти:

- Наявність 5 булетів.
- Зміст: чи описують переваги, чи відповідають товару.
- Відсутність змін Amazon (наприклад, скорочення через стоп-слова).

### Дії:

1. У Seller Central (**Inventory** > **Manage All Inventory** > **Edit** > **Product Details**) перевірте булети.
2. Оцініть, чи кожен пункт чітко описує перевагу чи характеристику.
3. Якщо булети неповні або змінені, створіть заявку (**Freelance** > **SEO**) для оновлення.
4. Повідомте старшого менеджера про зміни, внесені Amazon.

**Приклад**: Для ASIN "Bottle Holder" булети:

- Герметична кришка запобігає протіканню.
- Нержавіюча сталь, без BPA, безпечна для здоров’я.
- Зберігає напої гарячими 12 годин, холодними — 24 години.
- Компактні розміри 10x5x3 дюйми, зручно для подорожей.
- Золотий колір, стильний дизайн для щоденного використання.  
    Якщо булетів 3 або є стоп-слова, замовте оновлення.

> **Візуалізація**:  
> _Місце для GIF_: Додайте GIF (3–5 секунд), який показує редагування булетів у Seller Central.  
> _Опис_: Демонстрація, як перевірити та оновити булети.

## 3. Інтерактивний чекліст для тайтла та булетів

Використовуйте цей чекліст у Obsidian для відстеження прогресу:

- [ ]  Перевірено тайтл у Seller Central (≈200 символів, ключові слова).
- [ ]  Оцінено тайтл через Seller Sprite (довжина, SEO).
- [ ]  Перевірено 5 булетів (зміст, переваги).
- [ ]  Зрозуміло звіт із пропозиціями для менеджера.

## 4. Ключові поради для Account Manager'ів

- **Регулярність**: Перевіряйте тайтл і булети щотижня, щоб виявити зміни Amazon.
- **SEO**: Використовуйте Seller Sprite для аналізу ключових слів у тайтлі.
- **Чіткість**: Пишіть булети коротко, акцентуючи на перевагах для клієнта.
- **Заявки**: Для оновлення тайтла/булетів створюйте заявки (**Freelance** > **SEO**).
- **Комунікація**: Повідомляйте старшого менеджера про будь-які зміни, внесені Amazon.

> **Візуалізація**:  
> _Місце для GIF_: Додайте GIF, який показує перевірку тайтла та редагування булетів у Seller Central.  
> _Опис_: Демонстрація, як оптимізувати текстовий контент.
